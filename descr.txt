Build a complete, single-file HTML/CSS/JS frontend website (no backend required) for a placeholder luxury/premium company called "Lumière Co." All product and service names, descriptions, and images should be clearly marked as placeholder content so they can be swapped out later.

GENERAL REQUIREMENTS

Single self-contained HTML file with all CSS and JS written inline (no external files)
Fully responsive (mobile, tablet, desktop)
All placeholder text should use clearly labeled dummy content like [Service Name], [Product Description], etc. so it's easy to find and replace later
Use Google Fonts (import via link tag) — suggest a clean serif/sans-serif pairing appropriate for a luxury brand (e.g. Playfair Display + Inter)


DARK/LIGHT MODE TOGGLE

A toggle button fixed in the top left corner of the page
It should switch between a dark mode (deep charcoals, blacks, rich accent color like gold or champagne) and light mode (crisp whites, soft neutrals, same gold/champagne accent)
The toggle should use a smooth CSS transition across the entire page when switching
Use a sun/moon icon or similar intuitive icon to indicate the current mode
Default to dark mode on load


NAVIGATION BAR

Sticky/fixed top navbar with the company logo/name on the left
Nav links: About Us, Services, Goods, Customer Service, 🛒 Cart
The Cart icon should display a live item count badge that updates as items are added
Hover effects on nav links: smooth underline animation or subtle color shift
On mobile, collapse into a hamburger menu


HERO SECTION

Full-width animated hero section
Use a CSS gradient or animated background (e.g. subtle floating particles, gradient shift, or animated geometric shapes) — no external image dependencies
Large luxury headline (e.g. [Your Premium Headline Here]) with a subheading
A prominent CTA button with a hover glow or shimmer effect
Smooth fade-in animation on page load


SERVICES SECTION (id: services)

Display 3 placeholder services in a responsive card grid
Each card: icon (use a simple SVG or Unicode symbol), [Service Name], [Short Description], and a Learn More button
Card hover effect: lift/shadow effect + subtle border glow in the accent color
Cards should animate into view on scroll (use IntersectionObserver)


GOODS / PRODUCTS SECTION (id: goods)

Display 3 placeholder goods in a responsive card grid
Each card: placeholder image area (styled div with gradient fill), [Product Name], [Price], [Short Description], and an Add to Cart button
Add to Cart button: on click, increment the cart count in the navbar badge and give a brief visual confirmation (e.g. button flashes or shows "Added!")
Card hover effect: image area zooms slightly, card lifts, accent color border appears

A slide-in cart drawer/panel that opens from the right side when the cart icon in the navbar is clicked
Lists all added items with name, quantity, and price (placeholder prices like $[XX.XX])

Smooth slide-in/slide-out animation

3 placeholder testimonial cards in a horizontal scroll or grid layout

CUSTOMER SERVICE SECTION (id: customer-service)

A clean section with a placeholder contact form (Name, Email, Message fields + Submit button)

Images/media areas: subtle scale zoom
All hover transitions should be smooth (200–350ms ease)


SCROLL ANIMATIONS
